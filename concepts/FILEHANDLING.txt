🧠 What is it?

👉 Working with files:

read data

write data

update data

🔥 1) OPEN FILE
f = open("file.txt", "r")
Modes (must remember)
Mode	Meaning
"r"	read (default)
"w"	write (overwrite)
"a"	append
"r+"	read + write
⚡ 2) READ FILE
with open("file.txt", "r") as f:
    print(f.read())
Variations
f.read()        # full file
f.readline()    # one line
f.readlines()   # list of lines
⚡ 3) WRITE FILE
with open("file.txt", "w") as f:
    f.write("Hello Manoj")

👉 Creates file if not exists
👉 Overwrites if exists

⚡ 4) APPEND FILE
with open("file.txt", "a") as f:
    f.write("\nNew line")

👉 Adds data
👉 Does NOT delete old content

⚠️ IMPORTANT DIFFERENCE
"w" vs "a"
"w"	"a"
deletes old content ❌	keeps old content ✅
starts fresh	adds at end
🔥 5) BEST PRACTICE → with
with open("file.txt", "r") as f:
    data = f.read()

👉 Automatically closes file
👉 Prevents memory leaks

❌ Without with
f = open("file.txt", "r")
data = f.read()
f.close()

👉 Easy to forget close()

⚡ 6) REAL FLOW (IMPORTANT)
# Write
with open("data.txt", "w") as f:
    f.write("Python\n")

# Append
with open("data.txt", "a") as f:
    f.write("Manoj\n")

# Read
with open("data.txt", "r") as f:
    print(f.read())
Output
Python
Manoj
⚠️ COMMON ERRORS
1) File not found
open("abc.txt", "r")

👉 Error if file doesn’t exist

2) Overwriting mistake
open("file.txt", "w")

👉 Deletes old data ⚠️

"w" → overwrites the file (deletes old content and writes new)
"a" → adds new content at the end without deleting existing data
with open() is better because it automatically closes the file after use, even if an error occurs.
This prevents resource leaks and makes the code safer and cleaner.

🧠 INTERVIEW ANSWER

File handling in Python is used to read, write, and manipulate files using functions like open().
The with statement is preferred because it automatically handles file closing and avoids resource leaks.

🧠 What is JSON?

👉 Data format (like dictionary)

{
  "name": "Manoj",
  "age": 23
}
⚡ 1) Write JSON
import json

data = {"name": "Manoj", "age": 23}

with open("data.json", "w") as f:
    json.dump(data, f)
⚡ 2) Read JSON
with open("data.json", "r") as f:
    data = json.load(f)

print(data["name"])
⚡ Difference
Function	Use
json.dump()	write to file
json.load()	read from file
⚡ Real Use

APIs

Backend responses

Config files

🧠 What is CSV?

👉 Comma Separated Values
👉 Used in:

Excel

Data analysis

Real projects

⚡ Example CSV file
name,age
Manoj,23
John,25
🔥 1) Write CSV
import csv

data = [
    ["name", "age"],
    ["Manoj", 23],
    ["John", 25]
]

with open("data.csv", "w", newline="") as f:
    writer = csv.writer(f)
    writer.writerows(data)
🔥 2) Read CSV
import csv

with open("data.csv", "r") as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)
Output
['name', 'age']
['Manoj', '23']
['John', '25']
⚠️ Important

👉 CSV reads everything as string

⚡ Dictionary CSV (VERY USEFUL)
Write
import csv

data = [
    {"name": "Manoj", "age": 23},
    {"name": "John", "age": 25}
]

with open("data.csv", "w", newline="") as f:
    writer = csv.DictWriter(f, fieldnames=["name", "age"])
    writer.writeheader()
    writer.writerows(data)
Read
with open("data.csv", "r") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row["name"], row["age"])
🎯 Interview Answer

CSV is used to store tabular data. Python provides
the csv module to read and write CSV files using csv.reader,
csv.writer, and DictReader/DictWriter.

newline="" is used to prevent extra blank lines when writing CSV files, especially on Windows systems.
