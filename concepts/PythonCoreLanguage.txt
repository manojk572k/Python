Example: label numbers.

1️⃣ Counter

Counter counts occurrences of elements automatically.

Without Counter
freq = {}
nums = [1,2,2,3,3,3]

for n in nums:
    freq[n] = freq.get(n,0) + 1

print(freq)

Output

{1:1, 2:2, 3:3}
Using Counter
from collections import Counter

nums = [1,2,2,3,3,3]

freq = Counter(nums)

print(freq)

Output

Counter({3:3, 2:2, 1:1})

Much simpler.

Get Most Common Elements
from collections import Counter

nums = [1,2,2,3,3,3]

freq = Counter(nums)

print(freq.most_common(1))

Output

[(3,3)]

Meaning:

number 3 appeared 3 times
2️⃣ defaultdict

Normal dictionary throws error if key doesn’t exist.

Example:

d = {}

d["a"].append(1)

Error:

KeyError
Using defaultdict
from collections import defaultdict

d = defaultdict(list)

d["a"].append(1)
d["a"].append(2)

print(d)

Output

{'a':[1,2]}

It automatically creates the list.

Example: Group Words by First Letter
from collections import defaultdict

words = ["apple","ant","banana","bat"]

groups = defaultdict(list)

for w in words:
    groups[w[0]].append(w)

print(groups)

Output

{
 'a':['apple','ant'],
 'b':['banana','bat']
}
3️⃣ deque

deque = double ended queue

Fast insert/remove from both ends.

Normal list:

append() → fast
pop(0) → slow
Using deque
from collections import deque

d = deque([1,2,3])

d.append(4)
d.appendleft(0)

print(d)

Output

deque([0,1,2,3,4])
Removing Elements
d.pop()
d.popleft()
Rotate Example
from collections import deque

nums = deque([1,2,3,4])

nums.rotate(1)

print(nums)

Output

deque([4,1,2,3])
